# Deleting Objects with Drag and Drop

Direct manipulation to delete objects on web applications.

ドラッグ開始時に削除するためのドロップ領域を表示することで、OS 上でファイルをゴミ箱にドラッグアンドドロップして削除するのと同じような感覚を Web アプリケーション上で実現します。
